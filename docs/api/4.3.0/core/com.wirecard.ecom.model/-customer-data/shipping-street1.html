shippingStreet1

var shippingStreet1: String?

First line of the street address or equivalent local portion of the shipping address associated with the card use for this purchase. This field is limited to maximum 50 characters. This field is required unless shipping information is the same as billing information, or market or regional mandate restricts sending this information.
