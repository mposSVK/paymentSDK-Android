    Klarna¶

    Klarna payment modules serves as klarna android library wrapper and performs klarna payment using elastic engine.

    Supported Transaction Types¶

    This payment method supports only one transaction type:

    • authorization - places the account holder's funds on hold, pending future capture, re-authorization or void transaction.

    Klarna categories¶

    Supported categories:

    KLARNA_DEBIT - Klarna debit

    KLARNA_FINANCING - Klarna finance

    KLARNA_PAY_LATER - Klarna pay later

    KLARNA_TRANSFER - Klarna transfer

    Klarna order item types¶

    Klarna payment can consist of one or more order items. Every order item has its type. Some order item types add value to the total amount, another subtract total amount.

    Order item types that add value to the total amount:

    PHYSICAL - Physical item

    DIGITAL - Digital item

    GIFT_CARD - Gift card

    SHIPMENT_FEE - Shipment fee

    HANDING_FEE - Handling fee

    SALES_TAX - Sales tax

    Order item types that substract value to the total amount:

    DISCOUNT - Discount

    STORE_CREDIT - Store credit

    Signature¶

    Application shall get signature from server where signature shall be computed.

    Simple Klarna Payment¶

    Klarna requires additional setup of the wd_ecom_klarna_return_url_scheme and wd_ecom_klarna_return_url_host resources. An RuntimeException is thrown when these elements are missing.

    In order to receive response from klarna you need to add these xml string resouces in your project:

    ...

    <string name="wd_ecom_klarna_return_url_scheme">result.klarna</string>
    ...
</resources>

val klarnaPayment = KlarnaPayment(
        signature = signature,
        requestId = requestId,
        merchantAccountId = merchantAccountId,
        transactionType = TransactionType.AUTHORIZATION,
        // Amount = orderItemPhysical.amount(20) - orderItemDiscount.amount(5)
        amount = BigDecimal(15),
        currency = "EUR",
        category = Category.KLARNA_PAY_LATER,
        country = "AT",
        locale = "EN",
        orderItems = orderItems
)

Client(context = this, url = "https://api-test.wirecard.com", requestTimeout = 60).startPayment(klarnaPayment)

ArrayList<OrderItem> orderItems = new ArrayList<>();

KlarnaPayment klarnaPayment = new KlarnaPayment.Builder()
        .setSignature(signature)
        .setRequestId(requestId)
        .setMerchantAccountId(merchantAccountId)
        .setTransactionType(TransactionType.AUTHORIZATION)
        // Amount = orderItemPhysical.amount(20) - orderItemDiscount.amount(5)
        .setAmoundAndCurrency(new BigDecimal(15), "EUR")
        .setCategory(Category.KLARNA_PAY_LATER)
        .setCountry("AT")
        .setLocale("EN")
        .setOrderItems(orderItems)
        .build();

new Client(this, "https://api-test.wirecard.com").startPayment(klarnaPayment);

    Customizing Visuals¶

    To change colors within the SDK, override the default color values in the .xml file.

    Color Resources¶

    Changing any of these will affect every module used in the SDK:

    wd_ecom_color_main_light - header gradient start color

    wd_ecom_color_main - header gradient end color

    wd_ecom_color_toolbar_text - toolbar text and back arrow tint

    wd_ecom_color_main_background - background color for main view

    wd_ecom_color_pay_button_text - color of continue button

    wd_ecom_klarna_button_background - background color of continue button
